# Joint-Project

This project is creating a hypothetical dynamic e-commerce website, with specific focus on PHP and CSS skills.

## Overview
The finished website will allow authorized admins who login to upload products to the shop. Customers will be able to pretend buy those uploaded products.
The website will use an SQL database to store all the data and allow the website to be dynamic. 

## Technologies
- PHP
- HTML, CSS
- MySQL
- Github

## Setup Instructions
- Download repository files
- Open files in code editor
- Change SQL database config information to match your own
- Upload files to a server
- or use XAMMP with apache and MySQL
